# Cannon Shoot

Projeto de jogo da disciplina Estágio I

# Obejetivo

Este é um jogo onde a pessoa assume o controle sobre o canhão e esmaga 
todos os alvos que você pode encontrar.
Use seu dedo para apontar e colocar força.

O jogo possui:
* 10 níveis
* 2 Locais
* Música relaxante
